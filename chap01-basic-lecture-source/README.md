# chap01-basic-lecture-source

## 프로젝트 소개
Kotlin 기초 문법을 학습하기 위한 첫 번째 강의 자료입니다.

## 학습 내용

### 1. Main 함수
- Kotlin의 진입점인 `main()` 함수 사용법
- Java의 `public static void main(String[] args)`와의 차이점
- 매개변수 생략 가능 (Kotlin 1.3 이후)

### 2. 표준 출력
- `print()`: 줄바꿈 없이 출력
- `println()`: 줄바꿈과 함께 출력
- 한 줄에 여러 문장 작성 (세미콜론 구분 - 비권장)

### 3. 변수 선언
- `var`: 변경 가능한 변수 (mutable)

### 4. 표준 입력
- `readLine()`: 사용자로부터 입력 받기
- 문자열 연결 연산
