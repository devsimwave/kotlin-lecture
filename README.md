# Kotlin 강의 자료

Kotlin 프로그래밍 언어를 학습하기 위한 강의 자료 저장소입니다.

## 강의 구성

### [Chapter 01: Kotlin 기초](./chap01-basic-lecture-source)
Kotlin의 기본 문법과 기초 개념을 학습합니다.
- Main 함수
- 표준 입출력
- 변수 선언 (var)
- 문자열 연결

## 개발 환경
- Kotlin: 2.0.21
- JVM: 17
- Build Tool: Gradle

## 학습 방법
각 챕터별로 디렉토리가 구성되어 있으며, 각 챕터의 README.md 파일을 참고하여 학습할 수 있습니다.

```bash
# 특정 챕터로 이동
cd chap01-basic-lecture-source

# 프로젝트 실행
./gradlew run
```
