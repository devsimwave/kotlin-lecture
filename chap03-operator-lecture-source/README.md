# chap03-operator-lecture-source

## 학습 내용

### 1. 산술 연산자
- 기본 산술 연산자: `+`, `-`, `*`, `/`, `%`
- 정수 나눗셈: `6 / 5 = 1` (정수형 결과)
- 실수 나눗셈: `6.0 / 5.0 = 1.2` 또는 `toDouble()`로 변환
- 나머지 연산자: `%` (모듈로 연산)
- 복잡한 연산식: 괄호를 사용하여 우선순위 조정 가능
- Overflow/Underflow: 정수형 최대값을 초과하면 최소값으로, 최소값 미만이면 최대값으로 순환
- 대입 연산자: `=` 연산자로 값, 연산 결과, 표현식, 메소드 반환값 등 할당
- 복합 대입 연산자: `+=`, `-=`, `*=`, `/=`, `%=` (산술 연산과 대입을 동시에 수행)
- 증감 연산자: `++` (1 증가), `--` (1 감소)
- 부호 연산자: `+` (양수), `-` (음수, 부호 반전)

### 2. 비교 연산자와 논리 연산자
- 비교 연산자:
  - `==`: 같은지 비교 (동등 비교)
  - `!=`: 다른지 비교 (부등 비교)
  - `>`: 큰지 비교
  - `>=`: 크거나 같은지 비교
  - `<`: 작은지 비교
  - `<=`: 작거나 같은지 비교
  - 숫자, 문자열 등 다양한 타입 비교 가능
- 논리 연결 연산자:
  - `&&`: AND 연산 (둘 다 true일 때 true)
  - `||`: OR 연산 (하나라도 true면 true)
- 논리 부정 연산자:
  - `!`: NOT 연산 (true ↔ false 반전)

### 3. to 연산자
- Pair 객체 생성: `"key" to "value"`
- Map 생성 시 키-값 쌍 정의에 사용
- 예: `mutableMapOf("02" to "서울", "055" to "경남")`
- 인덱스 접근 연산자 `[ ]`: 맵의 키로 값 접근 (`map["key"]`)

### 4. in 연산자와 범위 연산자
- in 연산자:
  - 포함 여부 확인: 배열, 리스트, 집합, 범위(Range)에 요소 포함 여부 검사
  - 예: `3 in arr`, `'a' in list`
  - 부정 형태: `!in` (포함되지 않음)
  - 반복문에서 사용: `for (num in 1..5) { }`, `for (item in items) { }`
- 범위 연산자 `..`:
  - 범위 객체(Range) 생성
  - 예: `1..10` (1부터 10까지), `'a'..'z'` (a부터 z까지)
  - `in` 연산자와 함께 범위 검사에 사용
  - `for` 반복문에서 범위 순회에 사용
- `withIndex()`: 인덱스와 값을 함께 순회

### 5. 연산자 우선순위
- 높음 → 낮음 순서:
  1. 후위 연산자: `++`, `--`, `.`, `?.`, `?`
  2. 전위 연산자: `-`, `+`, `++`, `--`, `!`
  3. 타입 변환: `:`, `as`, `as?`
  4. 곱셈/나눗셈: `*`, `/`, `%`
  5. 덧셈/뺄셈: `+`, `-`
  6. 범위: `..`
  7. 엘비스: `?:`
  8. 포함 관계: `in`, `!in`, `is`, `!is`
  9. 대소 비교: `<`, `>`, `>=`, `<=`
  10. 동등 비교: `==`, `!=`
  11. AND: `&&`
  12. OR: `||`
  13. 대입/복합 대입: `=`, `+=`, `-=`, `*=`, `/=`, `%=`
